# Hybrid recommender app

> Simple real world Netflix like platform with built-in hybrid recommender

## Basic Setup

``` bash
# install dependencies
$ yarn

# serve client (dev)
$ yarn client:dev

# serve server (dev)
$ yarn server:dev

# build client for production and launch client app
$ yarn client:build
$ yarn client:start

# transpile server files and launch it
$ yarn server:start
```
